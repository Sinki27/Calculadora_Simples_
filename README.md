# 🧮 Calculadora Simples

Uma aplicação funcional e intuitiva para realizar operações matemáticas básicas diretamente no navegador. Este projeto foi desenvolvido para consolidar conceitos fundamentais de lógica de programação e manipulação de eventos.

---

## 🚀 Tecnologias Utilizadas

Este projeto foi construído com o "trio essencial" do desenvolvimento Web:

* **HTML5:** Estrutura dos botões e do visor da calculadora.
* **CSS3:** Design limpo e layout responsivo, garantindo que os botões sejam fáceis de usar em qualquer dispositivo.
* **JavaScript (ES6+):** Lógica por trás de todas as operações matemáticas e atualização em tempo real do visor.

---

## 🛠️ Funcionalidades

* **Operações Básicas:** Soma, subtração, multiplicação e divisão.
* **Interface Reativa:** Botões que respondem ao clique e atualizam o visor instantaneamente.
* **Função Limpar (C):** Reinicia o visor e as variáveis para uma nova conta.
* **Cálculos de Ponto Flutuante:** Suporte para números decimais.

---

## 🧠 O que eu pratiquei neste projeto?

1.  **Manipulação de DOM:** Captura de valores através dos cliques nos botões e exibição dinâmica dos resultados.
2.  **Lógica de Operadores:** Implementação de condições para garantir que as operações sigam a ordem correta.
3.  **Tratamento de Strings e Números:** Conversão de tipos de dados para realizar cálculos precisos a partir de entradas de texto.
